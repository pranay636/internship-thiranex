About TechStore

A simple E-commerce Product Catalog developed as a Web Development Capstone Project.

Project Overview

TechStore is a responsive web application developed using HTML, CSS and JavaScript. It demonstrates a modular frontend architecture with multiple pages, dynamic product rendering, client-side functionality and browser localStorage.

Features

  • Responsive User Interface
  • Product Catalog
  • Search Products
  • Category Filtering
  • Add to Cart
  • Remove from Cart
  • Persistent Cart using localStorage

Technologies Used

HTML5
CSS3
JavaScript
Local Storage
